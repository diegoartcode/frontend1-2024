Corpo HTML

Tag html

A tag html é o elemento raiz de um documento HTML e envolve todo o conteúdo da página. Ela indica ao navegador que o documento é um arquivo HTML e define o contexto para todo o conteúdo da página.


Tag head

A tag head contém metadados e informações sobre o documento HTML, como o título da página, links para folhas de estilo CSS, scripts JavaScript, codificação de caracteres e outras informações que não são diretamente exibidas na página.


Tag body

A tag body contém o conteúdo principal da página, incluindo texto, imagens, vídeos, links, formulários e qualquer outro elemento que seja visível para o usuário na janela do navegador.
