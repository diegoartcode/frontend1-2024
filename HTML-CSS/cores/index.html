Cores

Temos 4 tipos de cores em css.

De forma escrita exemplo color:red

De forma hexadecimal color:#ff0000

De forma RGB color:rgb(255,0,0), tendo uma variação que é a RGBA o A é de alfa. Exemplo rgba(255,0,0,0.5)

De forma HSL color:hsl(0,100%,50%), tendo uma variação que é a HSLA o A é de alfa. Exemplo hsl(0,100%,50%,50%)

RGB

Exemplo

rgb(255,171,205), red 255, green 171 e blue 205

R = red (vermelho)

G = Green (verde)

B = blue (Azul)

Hexadecimal

Exemplo

#FFABCC, red FF, green AB e blue CC

r = FF

g = AB

b = CC

rgb(255,186,204);

r = 255

g = 186

b = 204

HSL

Exemplo

hsl(0,100%,50%);

h = cor no círculo cromático

s = intensidade da saturação

l = claridade da cor
